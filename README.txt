<h1>Proyecto Final Modulo 1 - Desafío Latam </h1>


-En el siguiente proyecto, se solicita contruir un CV utilizando HTML y CSS, Bootstrap o algún template.
-Este debe contener una barra de navegación, header, secciones con Experiencia y Educación, Portafolio y Footer.
-Se debe despegar el proyecto en Github Pages.
-Se debe forkear el proyecto de 2 compañeros y realizar un commit por cada proyecto forkeado.


Estructura del directorio
-------------------
css\              - Archivos CSS de hojas de estilo.
images\           - Imágenes usadas en el html.
scripts\          - JavaScript utilizado en html.
styles\           - Archivos fuente SCSS para hojas de estilo CSS.
index.html        - Página HTML principal
*.html            - Páginas HTML adicionales (si las hay).
LICENSE-*.txt     - Archivo de licencia.
